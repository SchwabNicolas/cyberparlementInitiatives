Valider votre vote

Après votre vote, il est possible que l'on vous demande de le valider par email ou par SMS. Pour se faire, rendez-vous sur le service de messagerie.

Il faut alors recopier le code à 5 chiffres qui est fourni dans le message sur la page de validation. Si vous ne l'avez pas reçu, vous pouvez le renvoyer à l'aide d'un bouton prévu à cet effet.

Exemple d'email
noreply@cyberparlement.ch
